Typhone - программа для классификации звонков в ломбарде "Гарант".
1. Входные данные нужно сохранить в файле "IO\manual_input.txt".
Каждый звонок в отдельной строке.
2. Дата звонков записывается в виде "ДД.ММ.ГГ" на отдельной строке. Все записи под ней будут относиться к этой дате.
3. Затем нужно запустить скрипт invoke_typhone.py, который создаст чёрное окошко и сообщит в нём результаты своей работы.
Итоговый файл в формате Excel будет лежать в папке "IO".
